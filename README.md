# Greeting Mobile App

## Overview
This is a simple Android application that displays a greeting message based on user input.

## Features
- Enter a name
- Tap button
- Display greeting message

## Tech Stack
- Kotlin
- Android SDK
- XML Layouts

## How to Run
1. Open project in Android Studio
2. Start emulator
3. Click Run

## UI Elements
- Input field (EditText)
- Button
- TextView for result
